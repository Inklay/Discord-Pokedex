# Discord-Pokedex

# English

A Discord french Pokedex bot powered by Node.js

## Installation

### Windows
- Instal Node.js
- Clone the repository
- Locate the clonned repository and launch a terminal
- Run ```npm install```
- Enter your bot token in connect.js
- Run ```bot.bat```

### Linux and MacOS
- Instal Node.js
- Clone the repository
- Locate the clonned repository and launch a terminal
- Run ```npm install```
- Enter your bot token in connect.js
- Run ```bot.sh```


## Usage

In Discord type ```pokedex [Pokemon name]```
