# Discord-Pokedex

# English

A Discord french Pokedex bot powered by Node.js

## Installation

### Windows
- Clone the repository
- Locate the clonned repository and launch a terminal
- Run ```bot.bat```

### Linux and MacOS
- Clone the repository
- Locate the clonned repository and launch a terminal
- Run ```bot.sh```


## Usage

In Discord type ```pokedex [Pokemon name]```
